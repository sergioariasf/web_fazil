# Sergio Arias | Industrial IA Solutions Architect

Este es el repositorio profesional para mi portfolio como Arquitecto de IA Industrial. El sitio está diseñado para demostrar el puente entre la ingeniería de estructuras y la orquestación de IA Generativa.

## Estructura
*   `index.html`: Landing page principal con stack tecnológico y caso de estudio de ciclismo.
*   `privacy.html`: Política de privacidad (Garmin Connect compliant).
*   `styles.css`: Sistema de diseño basado en Tailwind CSS.

## Despliegue Automático (CI/CD)
Este proyecto está configurado para desplegarse automáticamente en **Cloudflare Pages** cada vez que se realiza un envío a la rama `main`.
